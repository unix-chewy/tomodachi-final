📖 Project Overview

Finals - Lab Quiz 2: Sentiment Analysis - NEGATIVE

Brief Description: This is a continuation of Finals - Lab Quiz 1 (Sentiment Analysis - POSITIVE), using the same LSTM-based RNN model deployed on Anvil. The web app takes user input text and returns either POSITIVE or NEGATIVE sentiment classification.

Objectives / Purpose: Demonstrate the sentiment analysis web app correctly identifying a NEGATIVE sentiment from user input.

Context Note: This uses the same model and Anvil deployment from Lab Quiz 1. The model was trained on the IMDB dataset and achieves reliable sentiment classification for both positive and negative reviews.

✨ Key Features

  • LSTM-based RNN: Captures sequential dependencies in text.
  • Anvil Web App: Simple text input interface for sentiment prediction.
  • Real-time Classification: Instant POSITIVE/NEGATIVE output.
  • Negative Sentiment Detection: Model correctly identifies negative movie reviews.
