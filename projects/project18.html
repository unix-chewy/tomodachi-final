📖 Project Overview

MIT App Inventor AI App Infographic

Brief Description: VAIson is an AI-powered mobile prototype built with MIT App Inventor that captures an image using the device camera and generates an AI-based description to help users understand visual content more easily. The app demonstrates practical integration of machine learning into a simple, accessible mobile interface.

Objectives / Purpose: Build a functional mobile app prototype that uses AI for image description, learn drag-and-drop coding for machine learning applications, and create an infographic documenting the project using NotebookLM.

✨ Key Features

  • Camera Integration: Capture images directly within the app.
  • AI-Based Image Description: Generate descriptive text from captured images.
  • Simple Interface: Easy-to-use buttons (Take Picture, Describe Image, CLEAR ALL).
  • Accessibility Focus: Helps users interpret visual scenes and supports accessibility needs.
  • Drag-and-Drop ML Coding: Built entirely using MIT App Inventor's visual programming environment.

🧩 Challenges & Solutions

Challenge 1: Implementing AI image description without traditional coding.

Solution: Used MIT App Inventor's visual blocks environment to connect camera output to AI service components through drag-and-drop.

Challenge 2: Understanding how machine learning concepts translate to block-based programming.

Solution: Focused on the workflow (capture → send → receive → display) rather than backend ML details, learning the practical integration layer of AI applications.

Challenge 3: Creating infographic visual given a short amount of time.

Solution: Used ChatGPT to generate an infographic summarizing the app's features, workflow, and impact.

🧠 Learning Reflection

What I learned: This project taught me that machine learning can be integrated into mobile apps even without traditional coding — drag-and-drop environments like MIT App Inventor make AI accessible to beginners. I learned how camera components, image handling, and AI description services work together in a practical application. This bridges the gap between theoretical ML concepts and real-world assistive technology.

Skills gained: Drag-and-drop programming in MIT App Inventor, integrating AI services into mobile apps, designing accessible user interfaces (Take Picture, Describe Image, CLEAR ALL), documenting projects with infographics using NotebookLM, and understanding the practical workflow of AI-powered mobile applications.
