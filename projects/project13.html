📖 Project Overview

Dogs vs. Cats Classifier

Brief Description: This project implements a Convolutional Neural Network (CNN) to classify images as either a dog or a cat. The model was trained on a dataset of dog and cat images and achieves accurate binary classification.

Objectives / Purpose: The primary objective is to test the trained Dogs vs. Cats binary classifier on two external images (one dog, one cat), correctly classify both, and capture screenshots of the prediction outputs.

✨ Key Features

  • Binary Classification: Model outputs probability between 0 (dog) and 1 (cat).
  • CNN Architecture: 3 convolutional layers (32, 64, 128 filters) with BatchNormalization and Dropout.
  • Activation Function: Sigmoid output layer for binary classification.
  • Preprocessing: Images resized to 128x128, normalized to [0,1].
  • TensorFlow Lite Export: Model converted to .tflite format for mobile deployment.
  • Flask Deployment Ready: Model saved and prepared for integration with Flask web framework.

💻 Important Code Blocks

1. CNN Model Architecture

3 convolutional layers with BatchNormalization, MaxPooling, and Dropout for regularization.


model = Sequential()
model.add(Conv2D(32, (3, 3), activation='relu', input_shape=(128, 128, 3)))
model.add(BatchNormalization())
model.add(MaxPooling2D(pool_size=(2, 2)))
model.add(Dropout(0.25))

model.add(Conv2D(64, (3, 3), activation='relu'))
model.add(BatchNormalization())
model.add(MaxPooling2D(pool_size=(2, 2)))
model.add(Dropout(0.25))

model.add(Conv2D(128, (3, 3), activation='relu'))
model.add(BatchNormalization())
model.add(MaxPooling2D(pool_size=(2, 2)))
model.add(Dropout(0.25))

model.add(Flatten())
model.add(Dense(512, activation='relu'))
model.add(BatchNormalization())
model.add(Dropout(0.5))
model.add(Dense(1, activation='sigmoid'))

model.compile(loss='binary_crossentropy', optimizer='rmsprop', metrics=['accuracy'])
model.load_weights("/content/model.h5")

2. Prediction Function

Loads and preprocesses an image, then returns the prediction score.


def get_prediction(test_img):
    test_img = test_img.resize((128, 128), resample=PIL.Image.BICUBIC)
    test_arr = img_to_array(test_img)
    test_arr = np.expand_dims(test_arr, axis=0)
    test_arr /= 255.0
    return float(model.predict(test_arr))

score = get_prediction(test_img)
if score < 0.5:
    print("It's a dog!")
else:
    print("It's a cat!")

🧩 Challenges & Solutions

Challenge 1: Model outputs a probability score; interpretation needed for user-friendly output.

Solution: Used threshold 0.5: score < 0.5 = dog, score > 0.5 = cat.

Challenge 2: Model weights needed to be loaded separately from architecture.

Solution: Used model.load_weights() to load pre-trained weights from model.h5 file.
