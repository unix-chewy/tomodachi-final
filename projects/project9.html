📖 Project Overview

Boston House Price Prediction Model Deployment Using Streamlit

Brief Description: This project deploys the trained Boston House Price Prediction model as an interactive web application using Streamlit, an open-source Python framework for building data apps quickly. Users can input house features (RM, LSTAT, PTRATIO) and receive instant price predictions.

Objectives / Purpose: The primary objective is to deploy the trained Boston House Price Prediction model as a Streamlit web application and capture a screenshot of the live app output as proof of successful deployment.

This current project builds upon that by deploying the trained model as a Streamlit web application.

💻 Important Code Blocks

Streamlit App Code

The complete Streamlit application code that loads the trained model and creates the interactive interface.


import streamlit as st
import numpy as np
import pickle
from tensorflow.keras.models import load_model

# Load trained model and scaler
model = load_model('boston_housing_model.h5')
scaler = pickle.load(open('scaler.pkl', 'rb'))

st.title("Boston House Price Predictor")
st.write("Enter the features below to predict the house price.")

# Input sliders
rm = st.slider("Average number of rooms (RM)", 3.0, 9.0, 7.5, 0.1)
lstat = st.slider("% lower status population (LSTAT)", 0.0, 40.0, 5.0, 0.1)
ptratio = st.slider("Pupil:Teacher ratio (PTRATIO)", 10.0, 25.0, 12.0, 0.1)

if st.button("Predict Price"):
    input_data = np.array([[rm, lstat, ptratio]])
    input_scaled = scaler.transform(input_data)
    prediction = model.predict(input_scaled)
    price_k = prediction[0][0]
    st.success(f"Predicted House Price: ${price_k:.2f}k")

🧩 Challenges & Solutions

Challenge: Ensuring the trained model and scaler are properly loaded in the Streamlit environment.

Solution: Saved the model as .h5 file and scaler as .pkl file, then loaded them at app startup.

Challenge: Displaying the output in a readable format (thousands of dollars).

Solution: Formatted the prediction as ${price:.2f}k to clearly indicate thousands of dollars.
