📖 Project Overview

Web App using CIFAR-10

Brief Description: This project deploys the trained CIFAR-10 CNN model as a web application, allowing users to upload images and receive real-time classification predictions across all 10 classes (airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck).

Objectives / Purpose: The primary objective is to demonstrate the deployed CIFAR-10 web app's multi-class classification capability by correctly predicting all 10 classes (airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck) and capturing screenshots as proof of accurate performance.

Context Note: This web app deployment is connected to Finals - Hands-on Exercise 2 (CIFAR-10 CNN Model Summary, Training and Accuracy), where the baseline CNN model was trained and evaluated.

✨ Key Features

  • 10-Class Classification: Model predicts all CIFAR-10 categories.
  • Image Upload Interface: Users can upload custom images for classification.
  • Real-time Prediction: Instant class label and confidence score displayed.
  • Web App Deployment: Flask-based interface for easy access.
