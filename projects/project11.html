📖 Project Overview

CIFAR-10 Highest Accuracy

Brief Description: This project builds upon the baseline CNN from Hands-on Exercise 2 to achieve the highest possible test accuracy on the CIFAR-10 dataset. Three improvement techniques were applied: increasing epochs, adding dropout layers, and experimenting with different optimizers and learning rates.

Objectives / Purpose: Improve model performance beyond the baseline accuracy of 77.76% by applying optimization techniques, with the goal of achieving the highest possible test accuracy.

Context Note: This is a continuation of Finals - Hands-on Exercise 2 (CIFAR-10 CNN Model Summary, Training and Accuracy). The baseline model from Exercise 2 achieved 77.76% test accuracy. This exercise focuses on improving that result.

✨ Key Features

  • Increased Epochs: Extended training from 20 to 30+ epochs to allow more pattern learning.
  • Dropout Regularization: Added Dropout layers (0.2-0.3) after convolutional and dense layers to reduce overfitting.
  • Optimizer Tuning: Experimented with Adam optimizer and adjusted learning rates.
  • Accuracy Improvement: Baseline: 77.76% → Improved: 78.41%.

🧩 Challenges & Solutions

Challenge 1: Baseline model showed signs of overfitting after extended training.

Solution: Added Dropout layers (0.2) after convolutional and dense layers to improve generalization.

Challenge 2: Determining optimal number of epochs — too few underfits, too many overfits.

Solution: Experimented with 25-35 epochs and monitored validation accuracy to find the sweet spot.

Challenge 3: Small accuracy improvement from baseline (only +0.65%).

Solution: Further improvements could include data augmentation (rotation, flipping, zoom) and batch normalization in future iterations.

🧠 Learning Reflection

What I learned: This exercise taught me that model improvement is an iterative process. Adding dropout regularization helped reduce overfitting, and increasing epochs allowed the model to learn more patterns. However, the improvement was modest (+0.65%), suggesting that more advanced techniques (data augmentation, batch normalization, learning rate scheduling) may be needed for significant gains.

Skills gained: Implementing dropout layers, tuning epochs for optimal performance, comparing baseline vs improved models, and documenting accuracy improvements.
