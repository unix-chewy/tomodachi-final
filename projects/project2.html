📖 Project Overview

Linear Regression Using Neural Network - Plant Growth

Brief Description: This project implements a Multi-layer Perceptron (MLP) Neural Network to model the non-linear relationship between temperature and plant growth. Unlike standard linear models, this neural network is designed to capture the quadratic nature of biological growth patterns, providing significantly more accurate predictions across the entire temperature spectrum.

Objectives / Purpose: The primary objective is show the Linear Regression given the notebook and document the completed project notebook into a PDF report. This includes organizing all code implementations, visualizations, and analytical findings into a single, well-formatted document suitable for submission, review, and future reference.

✨ Key Features

  • MLP Architecture: Features a dual hidden layer structure (50, 50) using the ReLU activation function to process complex data patterns.
  • Automated Pipeline: Implements an Scikit-learn Pipeline that integrates StandardScaler to ensure input features are normalized before reaching the neural network.
  • High-Iteration Training: Configured with the 'adam' solver and a maximum of 5,000 iterations to ensure optimal weight convergence.
  • Non-Linear Fitting: Successfully models a quadratic growth curve: $growth = -0.1 \times (temp - 25)^2 + 50$, even with added Gaussian noise.

💻 Important Code Blocks

1. Neural Network Pipeline

This pipeline ensures that data is scaled and then passed into an MLPRegressor with two hidden layers of 50 neurons each.


model = Pipeline([
    ("Scaler", StandardScaler()),
    ("nn", MLPRegressor(
        hidden_layer_sizes=(50,50),
        activation="relu",
        solver="adam",
        max_iter=5000,
        random_state=42
    ))
])

2. Performance Metrics Calculation

The project calculates MAE, MSE, and R² Score to quantitatively verify the model's accuracy.


mae = mean_absolute_error(growth, growth_pred)
mse = mean_squared_error(growth, growth_pred)

🧩 Challenges & Solutions

Challenge: Neural Networks are sensitive to the scale of input features (0-40°C), which can lead to slow convergence.

Solution: Wrapped the model in a StandardScaler pipeline to automatically normalize the temperature data before training.
