📖 Project Overview

CIFAR-10 CNN model Summary, Training and Accuracy

Brief Description: This project involves building a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset. The goal is to document the model architecture, training process, and accuracy results as a foundation for improving performance for the upcoming activities.

Objectives / Purpose: The primary objective is to build a Convolutional Neural Network (CNN) for CIFAR-10 image classification, document the model architecture summary, capture the training output over multiple epochs, and report the final test accuracy as a baseline for future model improvements

✨ Key Features

  • CNN Architecture: 4 convolutional layers (32→32→64→64 filters) with MaxPooling and Dropout.
  • Data Normalization: Pixel values scaled to [0,1] for faster convergence.
  • Training: 20 epochs with Adam optimizer (learning rate 0.0005).
  • Model Summary: Complete layer-by-layer architecture with trainable parameters.
  • Test Accuracy: Baseline accuracy recorded for future comparison.

💻 Important Code Blocks

1. CNN Model Definition

Baseline CNN architecture for CIFAR-10 classification.


model = tf.keras.models.Sequential()
model.add(Conv2D(32, 3, padding="same", activation="relu", input_shape=[32, 32, 3]))
model.add(Conv2D(32, 3, padding="same", activation="relu"))
model.add(MaxPool2D(pool_size=2, strides=2))
model.add(Conv2D(64, 3, padding="same", activation="relu"))
model.add(Dropout(0.2))
model.add(Conv2D(64, 3, padding="same", activation="relu"))
model.add(MaxPool2D(pool_size=2, strides=2))
model.add(Flatten())
model.add(Dense(128, activation='relu'))
model.add(Dropout(0.2))
model.add(Dense(10, activation='softmax'))

2. Model Compilation & Training


optimizer = tf.keras.optimizers.Adam(learning_rate=0.0005)
model.compile(loss="sparse_categorical_crossentropy",
              optimizer=optimizer, 
              metrics=["sparse_categorical_accuracy"])
model.fit(X_train, y_train, epochs=20)
