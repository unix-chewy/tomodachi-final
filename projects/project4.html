📖 Project Overview

Iris Classification Model Selection

Brief Description: This project implements a K-Nearest Neighbors (K-NN) classifier on the classic Iris dataset to predict flower species (Setosa, Versicolor, Virginica) based on sepal and petal measurements. The model was trained in Google Colab and deployed as an interactive web app using Anvil.

Objectives / Purpose: The primary objective is to deploy a functional Iris classification web app, then capture and submit three screenshots demonstrating correct predictions for each of the three Iris species (Setosa, Versicolor, and Virginica).

✨ Key Features

  • K-NN Classification: Uses K=10 neighbors to classify Iris species based on feature similarity.
  • Train-Test Split: 80% training / 20% testing split (random_state=4 for reproducibility).
  • Model Accuracy: Evaluates performance using accuracy_score.
  • Anvil Deployment: Exposes the trained model as a callable function (predict_iris) accessible via Anvil's server module.
  • Real-time Predictions: Users input sepal length/width and petal length/width to receive instant species prediction.

💻 Important Code Blocks

1. Load Dataset & Train-Test Split

Loads the Iris dataset and splits it into training and testing sets (80/20).


iris = load_iris()
X = iris.data
y = iris.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=4)

2. K-NN Model Training & Prediction

Initializes K-NN with 10 neighbors, fits the model, and makes predictions on test data.


knn = KNeighborsClassifier(n_neighbors=10)
knn.fit(X, y)
y_pred = knn.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)

3. Anvil Server Deployment

Connects to Anvil and creates a callable function for web app integration.


anvil.server.connect("server_key")
@anvil.server.callable
def predict_iris(sepal_length, sepal_width, petal_length, petal_width):
    classification = knn.predict([[sepal_length, sepal_width, petal_length, petal_width]])
    return iris.target_names[classification][0]
anvil.server.wait_forever()

This allows any Anvil frontend to call predict_iris and receive a species name.

🛠️ Tools & Technologies

  • Programming Language: Python 3.x

🧩 Challenges & Solutions

Challenge: Deciding the optimal K value for K-NN (K=10 chosen here, though default is 5).

Solution: While the code uses K=10 directly, a better approach would be to iterate through K values (1–20) and select the one with highest accuracy. This remains a future improvement.
