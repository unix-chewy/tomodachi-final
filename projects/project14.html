📖 Project Overview

Flask Web App Dogs vs. Cats Classifier

Brief Description: This project deploys the trained Dogs vs. Cats CNN model as a web application using Flask, a lightweight Python web framework. Users can upload images of dogs or cats through a browser interface and receive real-time classification results.

Objectives / Purpose: Deploy the pre-trained binary classification model to a Flask web app, allowing users to upload images and get predictions (dog or cat) with confidence scores and to upload 5 screenshots from the web output.

Context Note: This is the deployment phase of Finals - Lab Exercise 5 (Dogs vs. Cats Classifier), where the trained CNN model is integrated with a Flask backend to create a fully functional web application.

✨ Key Features

  • Flask Web Framework: Lightweight backend for handling image uploads and predictions.
  • Image Upload Interface: Users can select and upload images through a simple web form.
  • Real-time Classification: Model predicts dog or cat instantly upon upload.
  • Confidence Score Display: Shows prediction probability alongside the result.
  • Responsive Output: Displays uploaded image and classification result side by side.

💻 Important Code Blocks

Flask App Backend (app.py)

Main Flask application handling file uploads, preprocessing, and predictions.


from flask import Flask, request, render_template
from tensorflow.keras.models import load_model
from tensorflow.keras.preprocessing.image import load_img, img_to_array
import numpy as np
import os

app = Flask(__name__)
model = load_model('dogs_vs_cats_model.h5')

def preprocess_image(image_path):
    img = load_img(image_path, target_size=(128, 128))
    img_array = img_to_array(img)
    img_array = np.expand_dims(img_array, axis=0)
    img_array /= 255.0
    return img_array

@app.route('/', methods=['GET', 'POST'])
def index():
    if request.method == 'POST':
        file = request.files['file']
        if file:
            filepath = os.path.join('uploads', file.filename)
            file.save(filepath)
            img_array = preprocess_image(filepath)
            prediction = model.predict(img_array)[0][0]
            result = 'Dog' if prediction < 0.5 else 'Cat'
            confidence = prediction if prediction > 0.5 else 1 - prediction
            return render_template('result.html', result=result, 
                                   confidence=confidence * 100, 
                                   image_path=filepath)
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)
