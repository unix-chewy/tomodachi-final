📖 Project Overview

House Price Prediction

Brief Description: This is the deployment phase of the Boston House Price Prediction project. The trained Neural Network model was deployed as a fully functional web app using Anvil.works, allowing users to input house features (RM, LSTAT, PTRATIO) and receive real-time price predictions.

Objectives / Purpose: The primary objective is to deploy a functional Boston House Price Prediction web app and capture three screenshots showing successful predictions using three distinct test data inputs.

Context Note: This activity is an extended task from Midterm - Hands-on Long Quiz 1 (Boston House Price Prediction), where the model was originally developed and trained. The focus here is on deploying the model as a fully functional web application using Anvil.works and documenting its prediction capabilities.

✨ Key Features

  • Anvil Web App: Interactive form with three input fields (RM, LSTAT, PTRATIO).
  • Real-time Predictions: Model returns predicted house price instantly after user submits inputs.
  • Scaled Input Handling: Inputs are automatically scaled using the same StandardScaler from training.
  • Dollar Conversion: Predictions converted from scaled values to actual dollar amounts.
  • Test Data Predictions: Three separate screenshots showing predictions on different test inputs.

💻 Important Code Blocks

Anvil Server Connection & Prediction Function

This code runs in Google Colab and connects to Anvil.works, exposing the prediction function to the web app.


import anvil.server

anvil.server.connect("server_P5T2GXLUGFVFP3NUNWFDN2SK-DFICZZZXUTN4HAHY")

@anvil.server.callable
def predict_house_price(rm, lstat, ptratio):
    sample_input = np.array([[rm, lstat, ptratio]])
    sample_scaled = scaler.transform(sample_input)
    predicted_price = model.predict(sample_scaled)
    predicted_price_actual = predicted_price[0][0] * 1000
    return float(predicted_price_actual)

anvil.server.wait_forever()

The @anvil.server.callable decorator makes this function available to the Anvil frontend.
