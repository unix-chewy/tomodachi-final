📖 Project Overview

Linear Regression - Plant Growth

Brief Description: This project explores the application of a Linear Regression model to analyze the relationship between temperature and biological growth. Using a synthetic dataset that mimics real-world quadratic growth patterns, the model attempts to find the best linear fit to predict growth outcomes based on temperature inputs.

Objectives / Purpose: The primary objective is to document and compile the completed project notebook into a PDF report. This includes organizing all code implementations, visualizations, and analytical findings into a single, well-formatted document suitable for submission, review, and future reference.

✨ Key Features

  • Data Simulation: Utilizes NumPy to generate 500 temperature samples ranging from 0°C to 40°C, paired with growth data derived from a quadratic formula with Gaussian noise.
  • Model Implementation: Employs the Scikit-learn LinearRegression class to fit a predictive line to the generated temperature-growth dataset.
  • Quantitative Evaluation: Calculates and displays essential regression metrics including Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and the R² Score.
  • Visual Analysis: Includes a Matplotlib-based scatter plot that overlays the actual growth data with the model's linear prediction to visually highlight the fit.

💻 Important Code Blocks

1. Data Generation & Formula

This snippet defines the quadratic relationship between temperature and growth, serving as the ground truth for our experiment.


temp = np.linspace(0, 40, 500)
# Growth formula with noise: -0.1 * (temp - 25)^2 + 50
growth = -0.1 * (temp - 25) ** 2 + 50 + np.random.normal(0, 5, size=temp.shape)
temp = temp.reshape(-1, 1) # Reshaping for the model

2. Training and Prediction

The model is instantiated and fitted to the reshaped temperature data to generate growth predictions.


model = LinearRegression()
model.fit(temp, growth)

# Generate predictions across the temperature range
growth_pred = model.predict(temp)

🧩 Challenges & Solutions

Challenge: Fitting a straight line to data that naturally follows a curve, leading to higher error margins in the extremas (0°C and 40°C).

Solution: While Linear Regression provides a basic baseline, the evaluation metrics (specifically RMSE and R²) were used to quantify exactly how much variance the linear model failed to capture, providing a clear justification for moving toward polynomial regression in future iterations.
